-------- Funktionalität --------

-- Header  --
--> muss Sticky sein 
--> Deutsch / Englisch Switch + Übersetzung des Portfolios
--> Name klickbar führt zu About Me Sektion
--> About me / Skills / Portfolio klickbar und Slider führt zur richtigen Stelle

-- Footer --
--> Name muss klickbar sein, führt zum Anfang der Seite 
--> Legal Notice muss klickbar sein, führt zum Overlay Legal Notice
--> Icons (GitHub / LinkedIn führen zu Externen Seite in neuen Fenstern / Mail führt zur Contact Sektion)
_____________________________________________________________________________________________________

-- Front Page -- 
--> rechts Button Let's Talk klickbar / führt zur Contact Sektion
--> links Icons (GitHub / LinkedIn führen zu Externen Seite in neuen Fenstern / Mail führt zur Contact Sektion)
_____________________________________________________________________________________________________

-- My Skills --
--> Icon Continually learning bekommt Overlay bei Hover 
--> Get in touch Button klickbar, führt zur Contact me Sektion
______________________________________________________________________________________________________

-- Portfolio --
--> Button Live Test klickbar, neues Fenster mit Projekt öffnet sich 
--> Button GitHub klickbar, neues Fenster mit Git öffnet sich 
______________________________________________________________________________________________________

-- Comments -- 
--> Slider klickbar / Comments zu beiden Seiten slidebar
______________________________________________________________________________________________________

-- Contact -- 
--> ValidationForm funktionsfähig
--> Name / Email / Message REQUIRED eintragbar 
--> ohne Haken bei privacy Policy keine Mail verschicken möglich
--> Submit Button klickbar , send Mail / neues Fenster mit Bestätigung Overlay, damit keine Mails gespammt werden können
_______________________________________________________________________________________________________

-- Bottom --
--> Icon klickbar, scrollt an den Anfang der Seite 
_______________________________________________________________________________________________________

-- Legal Notice --
--> inkl. Close Button klickbar, Fenster schließt sich
_______________________________________________________________________________________________________

-- Privacy Policy --
--> inkl. Close Button klickbar, Fenster schließt sich
